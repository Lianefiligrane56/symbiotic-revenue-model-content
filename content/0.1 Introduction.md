## 2. Historic Data Computations

| Metric                                    | Definition                                                                                                                                                                                                                           |
| ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **TVL at Date ($)**                       | Total dollar value of all assets deposited/staked in Symbiotic protocol at a specific point in time. Measures protocol size, user confidence, and capital available for network security.                                            |
| **TVL v1 and v2**                         | Breakdown of Total Value Locked between Symbiotic's original protocol version (v1) and updated version (v2). Shows migration progress and which implementation users prefer.                                                         |
| **Cumulative Staking Rewards ($)**        | Total rewards earned by all stakers across ALL networks since protocol inception. Lifetime sum of every reward distributed through Symbiotic.                                                                                        |
| **Restaking Rewards ($)**                 | Rewards specifically from restaking activities where users stake already-staked assets (like stETH) to secure additional protocols simultaneously. Subset of Cumulative Staking Rewards showing capital efficiency.                  |
| **Network Security Fees (Hyperlane) ($)** | HYPER token rewards paid by Hyperlane Foundation to validators and stakers securing cross-chain messaging infrastructure. Token emissions (not revenue) used to incentivize network security - currently unsustainable.              |
| **L2s (Tanssi) ($)**                      | TANSSI token rewards paid by Tanssi Foundation to stakers securing their appchain/L2 deployment platform. Token emissions distributed to providers of economic security for rollup infrastructure.                                   |
| **Private Credit (Cap) ($)**              | Rewards paid to restakers providing insurance/credit guarantees for Cap protocol's lending operations. Derived from actual trading profits of operators (market makers, arbitrageurs) - sustainable real yield, not token emissions. |
| **Average APY (%)**                       | Average Annual Percentage Yield earned by stakers across all Symbiotic networks. Calculated as (Annual Rewards รท Average TVL) ร 100. Represents return on capital for security providers.                                            |
